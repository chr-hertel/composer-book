Prologue

Once upon a time we set out to create an open-source project.
It seemed like a small fun programming project, but eventually
it turned into this doorstopper, large enough to knock someone out.

This book will serve as a reminder that most software we rely
on is bug ridden, poorly documented, hard to comprehend at times,
and can easily grow into an untamable monstrosity if you're not
careful. So learn to say "No" a lot and don't put too much trust
into what you can read over the next couple hundred pages.

Despite all of this Composer is still an extremely
handy tool — quite contrary to this very book.

Nils Adermann
